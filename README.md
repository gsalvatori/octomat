Octomat
---


A C++ library to solve linear algebra problems.

Features:
---
Vector:

*size
*max
*push_back
*is_Empty
*clear
*fill_vector
*isEqual
*norm
*visualize
*operator +
*operator -
*operator * (double)
*operator * (Vector)

Compile:
---

Include octomat.cpp into your source file and compile it with:

* g++ -o test source.cpp

* ./test
